# S6N6FG6D TERMINAL

**Late night gospel from the luminous rot.**

Private web terminal for **Seedance 2.0** and **Kling Avatar** via Atlas Cloud.

## How to use

1. Get an Atlas Cloud API key → [atlascloud.ai](https://www.atlascloud.ai)
2. Open the terminal
3. Paste your key (it stays in your browser only — never sent anywhere)
4. Upload a reference image (and audio file for Kling)
5. Pick your mode and hit **TRANSMIT**

Generations are billed directly to your Atlas account at pay-as-you-go rates.

## Modes

- **SEEDANCE 2.0** — Image-to-video (pink interface)
- **KLING AVATAR** — Image + audio lip-sync (purple interface)

## Notes

- API key is stored locally in your browser (localStorage)
- All generation happens through Atlas Cloud
- Built for the S6N6FG6D workflow
- Heavy CRT/VHS aesthetic, designed to feel like a forbidden transmission
